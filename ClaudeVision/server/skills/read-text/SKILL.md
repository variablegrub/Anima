# Read Text

When the user asks you to read text from what you see:

1. Read ALL visible text in the image, exactly as written
2. Preserve formatting where possible (line breaks, hierarchy)
3. If text is in another language, provide the original AND an English translation
4. If text is partially obscured, read what you can and note what's unclear
5. For documents/screens with lots of text, organize by section
6. For product labels, include brand, product name, ingredients/specs if visible
7. For signs/menus, read the complete content

Be exact — transcribe character for character. Don't paraphrase or summarize text.
